Welcome to Elicit.

Elicit is an online platform designed for researchers to carry out online human subject experiments in the web browser. The project is mainly driven by researchers in collaboration with professional programmers that implement and expand the. It has been used to carry out several experiments and has now had over 1000 participant pass through. The platform is comprised of three parts,

  1. A frontend that visualize stimuli, captures user responses and interactions using scales and webcam eye tracking in the browser.
  2. An API that interacts with the database where user responses, behavioral data and interactions are stored for easy retrieval. A data model has been created that conceptually makes it easy to distinguish definition and results structure along with roles of each user (anonymized or registered).
  3. A client-side API (python) that allows researchers to both create and extract results from experiments.

To start creating experiments using this platform you would have to have basic knowledge about programming in Python as all interaction using the client-side API is in python. Otherwise, there is no need to know any other programming languages.

If you want to know more about the technical side of the platform, please visit https://github.com/elicit-experiment/overview

The platform was initially started at the Technical University of Denmark under the CoSound project. The focus was to carry out listening experiments, but the platform has since been expanded greatly using various funding sources and is now continued at the City College of New York. Investigators can now capture not only user responses through several scales but also capture eye movements using the WebGazer plugin.

The project is led by postdoc Jens Madsen and programmer Iain Bryson from Freeroaming solutions.
